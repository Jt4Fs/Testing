# How to Power a Computer

To power on a computer, follow these steps:

1. **Check the power source**: Ensure that the computer is plugged into a working power outlet.
2. **Locate the power button**: The power button is usually located at the front or top of the computer case.
3. **Press the power button**: Press the power button once to turn on the computer.
4. **Wait for the boot process**: After pressing the power button, the computer will take a few moments to boot up its operating system. 
5. **Log in**: Once the operating system loads, enter your credentials to log in.

## Troubleshooting
If the computer does not power on, check the following:
- Ensure the power cord is securely connected.
- Check if there is power to the outlet by testing with another device.
- If using a laptop, ensure the battery is charged or the power adapter is functioning properly.
